# Healthcare Management  Analytics

This repository demonstrates SQL skills applied to healthcare management. It includes projects focused on:

- Hospital capacity and bed management
- Appointment scheduling
- Doctor-patient relationships
- Clinical outcome analytics
- Data integrity and compliance (audit trail)

## Project Highlights
1. Emergency Case Success Rate – Calculate success rates by admission type.
2. Lab Procedures Analysis – Compare procedures across patient demographics.
3. Specialty Performance Analysis – Identify specialties with highest procedures per patient.
4. Automated Appointment Booking – Stored procedures to schedule patients.
5. Audit Trail – Triggers to track changes in patient records.

## Folder Structure
- `datasets/` : Sample CSVs for all entities
- `queries/` : SQL scripts for analysis and automation

## How to Use
1. Import CSVs into MySQL, PostgreSQL, or SQLite.
2. Run queries from `queries/` folder.
3. Visualize the database structure using the ER diagram.
4. Explore insights and adapt queries for practice or portfolio display.
